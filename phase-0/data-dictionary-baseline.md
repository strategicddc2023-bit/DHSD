## 2) ตาราง master ที่ต้องมีตั้งแต่ต้น
- master_agencies (สคร.1-สคร.13)
- master_provinces (77 จังหวัด)
- master_districts (ทุกอำเภอของไทย + province_code)

## 3) กฎสำคัญ
- เลือกจังหวัดแล้วต้อง query อำเภอเฉพาะจังหวัดนั้น

- เก็บทั้ง code และชื่อเพื่อประสิทธิภาพการแสดงผล
